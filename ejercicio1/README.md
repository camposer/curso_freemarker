Ejercicio #1
============

##Objetivo

Repasar los conceptos básicos de Freemarker

##Actividades

0. Cree una interfaz que permita gestionar Productos (CRUD+). Debe utilizar servlets, plantillas de freemarker y la clase de servicio implementada. 

Orden sugerido de trabajo:
- Listar productos
- Agregar
- Eliminar
- Mostrar
- Modificar

NOTA: En el PDF adjunto (ejercicio1.pdf) podrá conseguir un mockup simple de la interfaz.
